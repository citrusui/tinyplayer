TinyPlayer

A new music experience for your device!

Features

Light/Dark mode

In this mode, you can switch between the light and the dark mode when you want. The background is slightly blurred, which is pretty nice :)

Adaptive mode

If you know ColorFlow, this is the same concept. The background color and the color of the labels depends on the artwork. Adaptive mode = Limitless color possibilities!

Custom mode (Pro)

This is YOUR mode! You choose the background color and the color of the labels. Also, you can set the alpha of the backgroud that YOU want. Want more? Sure! You can set two modes in the custom mode! The only limit is your imagination!
